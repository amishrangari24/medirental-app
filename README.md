# 🚑 MediRental – Medical Equipment Rental App

## 📌 Problem Statement
Many patients face difficulty in finding affordable and essential medical equipment such as oxygen cylinders, wheelchairs, and hospital beds during emergencies. Existing rental services are unorganized, slow, and not easily accessible.

---

## 💡 Solution
MediRental is a mobile application that connects users with nearby vendors to rent medical equipment quickly, affordably, and efficiently. It simplifies the process of searching, booking, and receiving medical equipment at home.

---

## 🚀 Features
- 🔍 Search medical equipment
- 📍 Location-based availability
- 📦 Easy rental booking
- 🚚 Delivery tracking (basic)
- 👤 User profiles
- 💳 Payment options (Cash on Delivery / UPI)
- 🚨 Emergency quick access button

---

## 🛠️ Tech Stack
- **Frontend:** Flutter
- **Backend:** Firebase
- **Database:** Firestore
- **APIs:** Google Maps API

---

## 📱 App Flow
1. User opens the app  
2. Searches for required equipment  
3. Selects nearby vendor  
4. Chooses rental duration  
5. Confirms booking  
6. Vendor accepts request  
7. Delivery is initiated
